# ToDoApp2

This is my **ToDoApp Android project** built using **Jetpack Compose**.  
It allows users to create, edit, and delete tasks easily with a modern and responsive UI.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Modern UI using Jetpack Compose
- Data persistence using local storage (Room / SharedPreferences)
- Clean and modular code structure

## Tech Stack

- **Language:** Kotlin  
- **UI:** Jetpack Compose  
- **Architecture:** MVVM
- **Storage:** Room / SharedPreferences
